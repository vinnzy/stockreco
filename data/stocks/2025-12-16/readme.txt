National Stock Exchange of India Limited
Capital Market Information Service 

Information related to  files in nuprddmmyy.zip (extranet file name)/PRddmmyy.zip (website file name):

This is to inform all the news-vendors and news agencies that, on NSE various instruments like Equities, Debentures, Warrants, Partly Paid Shares etc. are traded.

The nuprddmmyy.zip (extranet file name)/PRddmmyy.zip (website file name) contains the following files:

anddmmyyyy.txt
bcddmmyyyy.csv
bhddmmyyyy.csv
bmddmmyyyy.txt
corpbondddmmyyyy.csv
etfddmmyyyy.csv
glddmmyyyy.csv
hlddmmyyyy.csv
pdddmmyyyy.csv
prddmmyyyy.csv
readme.txt
smeddmmyyyy.csv
ttddmmyyyy.csv
mcapddmmyyyy.csv


anddmmyyyy.txt file contains name of the co., its symbol and the announcements made during the day.    

bcddmmyyyy.csv file contains the corporate action details for securities.

bhddmmyyyy.csv file contains a list of securities which have hit their price bands during the day.

bmddmmyyyy.txt file contains details of Board Meetings of companies

corpbondddmmyyyy.csv file contains market data of Corporate bonds traded during the day.

etfddmmyyyy.csv file contains market data of exchange traded funds traded during the day.

glddmmyyyy.csv file contains a list of the gainers and losers for Nifty Securities, Nifty Next 50 Securities and for other Securities.

hlddmmyyyy.csv file contains the list of securities (Equity and ETFs) which have reached a new high or a new low.

pdddmmyyyy.csv file also contains Symbol and Series codes for each Security in addition to the information contained in the prddmmyyyy.csv file. 52 week high & 52 week low prices are not adjusted for Bonus, Split & Rights corporate actions

prddmmyyyy.csv file contains security wise market data information along with the Index details. 52 week high & 52 week low prices are not adjusted for Bonus, Split & Rights corporate actions

readme.txt is a informative file which contains information about data and structure of all other files.

smeddmmyyyy.csv file contains market data of SME securities traded during the day.

ttddmmyyyy.csv file contains a list of top twenty five securities by traded value.

mcapddmmyyyy.csv file contains market capitalization of all the listed and permitted companies of the exchange by using latest available price.

Note : 52 week high / 52 week low and other High / Low prices mentioned in the above files are not adjusted for corporate actions.
